# Components

## Layout
- `src/components/app-shell.tsx`: Wrapper layout with header and page container.
- `src/components/hero.tsx`: Intro section and CTA links on home page.

## Topic Cards & Sections
- `src/components/topic-card.tsx`: Topic summary, status badge, progress bar.
- `src/components/topic-header.tsx`: Topic title/description header.
- `src/components/topic-outline.tsx`: Theory list block.
- `src/components/example-block.tsx`: Code example block.

## Exercises
- `src/components/exercise-list.tsx`: Exercise rendering + hint tooltip + solution.
- `src/components/locked-overlay.tsx`: Locked state UI for gated topics.
- `src/components/progress-overview.tsx`: Overall progress summary on home.

## UI Toolkit
shadcn components used from `src/components/ui/`:
- `button`, `card`, `badge`, `progress`, `tooltip`, `separator`, `tabs`
